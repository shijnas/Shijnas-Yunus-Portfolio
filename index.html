Interactive QA Sandbox

BUG-104 Priority: High Severity: Medium

Character gets stuck while entering a vehicle

Build: v1.0.4-live
Platform: Windows PC
Category: Animation / Logic

Steps to Reproduce:

  1. Spawn a standard 4-door sedan.
  2. Stand directly next to the driver's side door.
  3. Press the enter key F repeatedly while moving.
  4. Observe player model freezing mid-animation.
✓ Expected Result

Character initiates entry animation and smoothly enters the driver's seat.

✗ Actual Result

Player model freezes. Controls lock out. Character remains stuck until reconnecting.
